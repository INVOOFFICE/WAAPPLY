Beta ? MenteDB is under active development. The Mind Database for AI Agents MenteDB is a purpose built database engine for AI agent memory. Not a wrapper around existing databases, but a ground up Rust storage engine that understands how AI/LLMs consume data. mente (Spanish): mind, intellect Quick Start Just remember a conversation: # Via REST API curl -X POST http://localhost:6677/v1/ingest \ -H " Content-Type: application/json " \ -d ' {"conversation": "User: I prefer Python over JS\nAssistant: Noted!", "agent_id": "my-agent"} ' # Response: { "memories_stored": 2, "rejected_low_quality": 5, "contradictions": 0 } Via MCP (Claude CLI, Copilot CLI, Cursor): // claude_desktop_config.json { "mcpServers" : { "mentedb" : { "command" : " mentedb-mcp " , "args" : [ " --data-dir " , " ~/.mentedb " ] } }
} Then the AI can call ingest_conversation directly. Embed in Rust: use mentedb :: MenteDb ; let db = MenteDb :: open ( "./my-agent-memory" ) ? assemble_context ( agent_id , space_id , 4096 ) ? Every database ever built assumes the consumer can compensate for bad data organization. A transformer gets ONE SHOT, a single context window, a single forward pass. MenteDB is a cognition preparation engine that delivers perfectly organized knowledge because the consumer has no ability to reorganize it. The Memory Quality Problem Most AI memory tools store everything and retrieve by similarity. The result: context windows full of noise. Studies show up to 97% of automatically stored memories are irrelevant. MenteDB solves this with write time intelligence: LLM Powered Extraction parses conversations and extracts only what matters: decisions, preferences, corrections, facts, entities Entity-Centric Memory extracts structured entities (people, pets, places, events) with typed attributes and links them to related memories via knowledge graph edges ? so "bought a collar for my Golden Retriever" remembers the breed, not just the collar Quality Filtering rejects low confidence extractions before they hit storage Deduplication checks embedding similarity against existing memories Contradiction Detection flags when new information conflicts with existing beliefs Belief Propagation cascades updates when facts change The result: a clean, curated memory that actually helps the AI perform better.
