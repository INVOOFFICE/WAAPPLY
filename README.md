# waapply (Artificial Intelligence News)

This folder contains a complete Artificial Intelligence news blog frontend.

Data source (generated by Google Apps Script):
- `ai-news/news.json` (source of truth: `{ site, articles[] }`)

Static generation (SEO pages):
- Run `node ai-news/scripts/build-ai-news-pages.mjs` to generate:
  - `ai-news/articles/<slug>/index.html`
  - `ai-news/sitemap.xml`

Automation:
- Use `ai-news/ai-news-blog/code.gs` inside your Google Sheet Apps Script project.
- The script fetches Artificial Intelligence news from free APIs, deduplicates, enriches SEO fields, exports `news.json` + `sitemap.xml`, and pushes to GitHub.
